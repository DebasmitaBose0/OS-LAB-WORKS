Question 1:

Write a shell program to find the average of three floating point numbers.

Code:
read -p "Enter three numbers: " a b c

avg=$(echo "scale=2; ($a+$b+$c)/3" | bc)

echo "Average = $avg"

Output:
Enter three numbers: 2.5 3.5 4.0
Average = 3.33

Question 2:

Write a shell program to find the area of a circle.

Code:
read -p "Enter radius: " r

area=$(echo "scale=2; 3.14*$r*$r" | bc)

echo "Area of circle = $area"

Output:
Enter radius: 5
Area of circle = 78.50

Question 4:

Write a shell program to find largest among three numbers.

Code:
read -p "Enter three numbers: " a b c

if [ $a -ge $b ] && [ $a -ge $c ]; then
    echo "$a is largest"
elif [ $b -ge $a ] && [ $b -ge $c ]; then
    echo "$b is largest"
else
    echo "$c is largest"
fi

Output:
Enter three numbers: 4 9 2
9 is largest

Question 5:

Write a shell program to check if a year is leap year or not.

Code:
read -p "Enter year: " y

if [ $((y % 400)) -eq 0 ] || [ $((y % 4)) -eq 0 -a $((y % 100)) -ne 0 ]; then
    echo "Leap year"
else
    echo "Not a leap year"
fi

Output:
Enter year: 2024
Leap year

Question 6:

Write a shell program to check whether an alphabet is a vowel or consonant.

Code:
read -p "Enter a character: " ch

case $ch in
a|e|i|o|u|A|E|I|O|U)
    echo "Vowel"
    ;;
*)
    echo "Consonant"
    ;;
esac

Output:
Enter a character: a
Vowel
